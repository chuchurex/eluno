# PROMPT: Construir Índice Temático de Sesiones Ra

> Este prompt se ejecuta UNA VEZ para generar un índice reutilizable.
> El resultado se guarda como `RA_THEMATIC_INDEX.md` y se incluye en el proyecto.

---

## OBJETIVO

Construir un índice temático exhaustivo de las sesiones de Ra (1-106) organizado por los temas de los 16 capítulos del libro "El Uno". El índice mapea cada tema a las sesiones.preguntas exactas donde Ra lo aborda.

---

## FUENTES

Busca en estos archivos .md:
- `the_ra_contact_volume_1.md` — Sesiones 1-56
- `the_ra_contact_volume_2.md` — Sesiones 57-106

El formato de cada intercambio en los .md es:
```
**XX.YY** QUESTIONER [pregunta]
R A I am Ra. [respuesta]
```


### Reglas:
1. Usa notación SESIÓN.PREGUNTA (ej: 13.5, no "session 13 question 5")
2. Incluye nota breve indicando dónde está el tratamiento más completo
3. Si una sesión.pregunta es relevante para múltiples capítulos, inclúyela en todos
4. Busca activamente en los .md — no confíes solo en memoria
5. Prioriza pasajes donde Ra explica conceptos directamente, no menciones tangenciales
