# Ver Roadmap del Proyecto

Muestra el estado actual del roadmap y tareas pendientes.

## Instrucciones

1. Lee el archivo `docs/project/ROADMAP.md` del repositorio eluno-org-archive
2. Extrae y muestra:
   - Fase actual del proyecto
   - Tareas completadas (marcadas con [x])
   - Tareas pendientes (marcadas con [ ])
   - Próximos hitos

3. Calcula progreso por fase:
   ```
   Phase 1: ████████░░ 80%
   Phase 2: ██░░░░░░░░ 20%
   ```

4. Destaca las 3 tareas más urgentes según prioridad

## Ejemplo de uso
`/project:roadmap` - Muestra estado del roadmap
